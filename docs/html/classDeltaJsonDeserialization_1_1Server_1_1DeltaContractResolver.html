Detailed Description

This class implements a JsonContractResolver to provide support for deserialization of the Delta<T> type using Json.NET.

See WebApiConfig.cs to see how the this class is used in the configuration.

The contract created for Delta<T> will deserialize properties using the types and property names of the underlying type. The JsonProperty instances are copied from the underlying type's JsonContract and customized to work with a dynamic object. In particular, a custom IValueProvider is used to get and set values using the contract of DynamicObject, which Delta<T> inherits from.
