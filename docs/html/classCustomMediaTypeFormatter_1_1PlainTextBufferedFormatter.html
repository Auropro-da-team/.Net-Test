Detailed Description

This sample formatter illustrates how to use the BufferedMediaTypeFormatter base class for writing a MediaTypeFormatter. The BufferedMediaTypeFormatter is useful when you either want to aggregate many small reads or writes or when you are writing synchronously to the underlying stream.
