The methods provided in this section are simply used to allow NavigationHelper to respond to the page's navigation methods.

Page specific logic should be placed in event handlers for the
GridCS.Common.NavigationHelper.LoadState and GridCS.Common.NavigationHelper.SaveState. The navigation parameter is available in the LoadState method in addition to page state preserved during an earlier session.
