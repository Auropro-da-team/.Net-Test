Parameters
segmentCan be empty, a target prefix value, or a route parameter name embeded in "{" and "}".

If segment is null or empty, the parameter will match all color values from the whole path. If segment is "oranges", the parameter will match color only from the segment starting with "oranges" like .../oranges;color=red/... If segment is "{fruits}", the parameter will match color only from the route .../{fruits}/...
