Detailed Description

Used to bind the segment prefix value from the route.

If [Route["{fruits}/{location}"] is specified and the incoming uri's relative path is "/apples:color=red,green/washington;rate=good", then in the action's argument list, [SegmentPrefix] string fruits will have fruits = apples but string location without this attribute will have location = washington;rate=good.
