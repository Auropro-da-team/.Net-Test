Detailed Description

This demonstrates the use of a controller-level attribute route. Notes:

  • Here the controller-level route applies to all actions except 'GetDetailsOfOrder'. i.e. Route attributes defined on an action override controller-level routes.
  • The 'PostOrder' action uses the controller-level route name to generate links.
