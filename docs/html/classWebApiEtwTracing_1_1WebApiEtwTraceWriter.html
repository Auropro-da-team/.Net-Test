Traces the specified information if permitted.

Parameters
requestThe HttpRequestMessage to associate with this trace.
categoryA string specifying an arbitrary category name for the trace.
levelThe TraceLevel of the trace.
traceActionAn action to call for the detailed trace information if this method determines the trace request will be honored. The action is expected to fill in the given TraceRecord with any information is wishes.

This is the public entry point for tracing via ITraceWriter, and it is called by both core WebAPI code as well as user code. This implementation will use TraceRequestFilter and TraceRecordFilter to determine whether to trace the information or discard it.

Func<TraceRecord, bool> WebApiEtwTracing.WebApiEtwTraceWriter.TraceRecordFilter
getset

Gets or sets a predicate to determine whether a specific TraceRecord should be written to ETW.

This predicate is executed only after the TraceRequestFilter predicate has returned true to accept the trace request. This predicate can examine the fully-formed TraceRecord to determine whether it should be written to ETW. If the predicate returns true the TraceRecord will be written to ETW, otherwise it will be discarded. A null value for this predicate allows all TraceRecord instances to be written.

Func<TraceLevel, string, bool> WebApiEtwTracing.WebApiEtwTraceWriter.TraceRequestFilter
getset

Gets or sets a predicate to determine whether an individual trace request should be accepted.

The default value for this predicate allows only a subset of traces to be accepted, including the incoming request, outgoing response, and any warnings or errors. If this predicate returns true the trace request will be accepted, otherwise it will be discarded. A null value for this predicate allows all trace requests.
