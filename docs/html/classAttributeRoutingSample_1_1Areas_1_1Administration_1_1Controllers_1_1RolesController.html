Detailed Description

NOTE: In this Administration area, we have the following setup:

  • UsersController uses conventional routes defined in AdminAreaRegistration.cs
  • RolesController uses attribute routing

For this kind of setup to work correctly with the rest of the application, MVC's attribute routing should be registered in an appropriate place. You can take a look at the Global.asax.cs file for the route order that is recommended.
