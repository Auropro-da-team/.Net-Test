Detailed Description

This controller mixes conventional and attribute routing in a single controller. The 'Professor' action uses attribute routing while rest of the actions are accessible by conventional routes.

The comments by each action detail example URLs that could reach the action.

NOTE: Conventional routes can never access attributed controller/actions
