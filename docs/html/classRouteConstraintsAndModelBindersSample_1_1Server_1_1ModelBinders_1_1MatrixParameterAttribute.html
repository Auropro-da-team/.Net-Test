Parameters
segmentCan be empty, a target prefix value, or a general route name embeded in "{" and "}".

[MatrixParam("")] string[] color will match all color values from the whole path. [MatrixParam("oranges")] string[] color will match color only from the segment starting with "oranges" like .../oranges;color=red/... [MatrixParam("{fruits}")] string[] color will match color only from the route .../{fruits}/...
