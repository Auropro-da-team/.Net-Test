\doxysubsection{Detailed Description}
This controller uses attribute routing to map the Index action to several different URL paths. 

The \mbox{[}Route\+Prefix\mbox{]} attribute defines a prefix for all routes defined on the controller and on on actions. The \mbox{[}Route\+Prefix\mbox{]} attribute on its own does not create any route entries, you must add \mbox{[}Route\mbox{]} attributes to create routes.

In this example the \mbox{[}Route\mbox{]} attribute is used with no arguments to specify that \textquotesingle{}/\+Home\textquotesingle{} should match the index action.

The \mbox{[}Route\mbox{]} attribute is also used on the Index action with \textquotesingle{}Index\textquotesingle{}, this will be combined with the \textquotesingle{}Home\textquotesingle{} prefix to match \textquotesingle{}/\+Home/\+Index\textquotesingle{}

To override a prefix, specify the \textquotesingle{}$\sim$/\textquotesingle{} qualifier, and then the full path that should match that route. In this example \textquotesingle{}$\sim$/\textquotesingle{} is used to specify a route that matches the root request path \textquotesingle{}/\textquotesingle{}.
