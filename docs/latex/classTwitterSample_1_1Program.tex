The sample uses a Http\+Message\+Handler to insert the appropriate \mbox{\hyperlink{namespaceTwitterSample_1_1OAuth}{OAuth}} authentication information into the outgoing Http\+Request\+Message. The result from twitter is read as a JToken. 

Before you can run this sample you must obtain an application key from twitter, and fill in the information in the OAuth\+Message\+Handler class, see \href{http://dev.twitter.com/}{\texttt{ http\+://dev.\+twitter.\+com/}} for details.
