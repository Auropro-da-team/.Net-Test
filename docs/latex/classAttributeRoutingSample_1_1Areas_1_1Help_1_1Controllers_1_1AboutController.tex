\doxysubsection{Detailed Description}
This controller uses attribute routing inside an Area. 

The \mbox{[}Route\+Area\mbox{]} attribute specifies a prefix for this area, and replaces the \textquotesingle{}Area\+Config\textquotesingle{} class that an area would typically require when using conventional routing. For URL matching purposes, the \mbox{[}Route\+Area\mbox{]} attribute is similar to the \mbox{[}Route\+Prefix\mbox{]} attribute.

The \mbox{[}Route\mbox{]} attribute defined on the controller uses a default value for the \{action\} parameter. In this case the default is \textquotesingle{}Index\textquotesingle{}, so this route can match patterns like both \textquotesingle{}/\+Help/\+About\textquotesingle{} and \textquotesingle{}/\+Help/\+About/\+Index\textquotesingle{}. Note that the prefix specified by the \mbox{[}Route\+Area\mbox{]} attribute is combined with the template.
