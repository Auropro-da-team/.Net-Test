This demonstrates the use of a controller-\/level attribute route. Notes\+: 


\begin{DoxyItemize}
\item Here the controller-\/level route applies to all actions except \textquotesingle{}Get\+Details\+Of\+Order\textquotesingle{}. i.\+e. Route attributes defined on an action override controller-\/level routes.

\end{DoxyItemize}
