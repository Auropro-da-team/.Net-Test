\doxysubsection{Detailed Description}
Navigation\+Manager aids in navigation between pages. It provides commands used to navigate back and forward as well as registers for standard mouse and keyboard shortcuts used to go back and forward. In addition it integrates Suspension\+Manger to handle process lifetime management and state management when navigating between pages. 

To make use of Navigation\+Manager, follow these two steps or start with a Basic\+Page or any other Page item template other than Blank\+Page.

1) Create an instance of the Naivgation\+Helper somewhere such as in the constructor for the page and register a callback for the Load\+State and Save\+State events.
