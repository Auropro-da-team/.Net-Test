\doxysubsection{Detailed Description}
Here we demonstrate the following\+: 


\begin{DoxyItemize}
\item Decorate attribute route on each action.
\item Escaping Route\+Prefix by using \textquotesingle{}$\sim$/\textquotesingle{}
\item Multiple Route attributes on an action
\item Uri\+Path\+Extension\+Mapping with attribute routing
\end{DoxyItemize}

NOTE\+:
\begin{DoxyItemize}
\item Route\+Prefix never creates a route in the route table, where as Route\+Attribute does
\item This example is a verbose way of using attribute routing where each action is attributed explicitly. You can take a look at the \mbox{\hyperlink{classWebApiAttributeRoutingSample_1_1Controllers_1_1Api_1_1CustomersController}{Customers\+Controller}} \& \mbox{\hyperlink{classWebApiAttributeRoutingSample_1_1Controllers_1_1Api_1_1OrdersController}{Orders\+Controller}} for better ways of improving this controller.
\end{DoxyItemize}
