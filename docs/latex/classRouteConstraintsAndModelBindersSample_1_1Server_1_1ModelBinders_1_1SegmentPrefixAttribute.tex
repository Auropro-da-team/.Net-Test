\doxysubsection{Detailed Description}
Used to bind the segment prefix value from the route. 

If \mbox{[}Route\mbox{[}\char`\"{}\{fruits\}/\{location\}\char`\"{}\mbox{]} is specified and the incoming uri\textquotesingle{}s relative path is \char`\"{}/apples\+:color=red,green/washington;rate=good\char`\"{}, then in the action\textquotesingle{}s argument list, {\ttfamily \mbox{[}Segment\+Prefix\mbox{]} string fruits} will have fruits = apples but {\ttfamily string location} without this attribute will have location = washington;rate=good.
