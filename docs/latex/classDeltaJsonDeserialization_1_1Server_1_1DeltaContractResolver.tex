\doxysubsection{Detailed Description}
This class implements a Json\+Contract\+Resolver to provide support for deserialization of the Delta$<$\+T$>$ type using Json.\+NET. 

See Web\+Api\+Config.\+cs to see how the this class is used in the configuration.

The contract created for Delta$<$\+T$>$ will deserialize properties using the types and property names of the underlying type. The Json\+Property instances are copied from the underlying type\textquotesingle{}s Json\+Contract and customized to work with a dynamic object. In particular, a custom IValue\+Provider is used to get and set values using the contract of Dynamic\+Object, which Delta$<$\+T$>$ inherits from.
