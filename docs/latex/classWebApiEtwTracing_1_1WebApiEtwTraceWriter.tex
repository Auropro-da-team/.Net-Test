Gets or sets a predicate to determine whether a specific Trace\+Record should be written to ETW. 

This predicate is executed only after the \mbox{\hyperlink{classWebApiEtwTracing_1_1WebApiEtwTraceWriter_ae4ff015ec700bf4e7972e8fab3d74383}{Trace\+Request\+Filter}} predicate has returned {\ttfamily true} to accept the trace request. This predicate can examine the fully-\/formed Trace\+Record to determine whether it should be written to ETW. If the predicate returns {\ttfamily true} the Trace\+Record will be written to ETW, otherwise it will be discarded. A {\ttfamily null} value for this predicate allows all Trace\+Record instances to be written. \mbox{\Hypertarget{classWebApiEtwTracing_1_1WebApiEtwTraceWriter_ae4ff015ec700bf4e7972e8fab3d74383}\label{classWebApiEtwTracing_1_1WebApiEtwTraceWriter_ae4ff015ec700bf4e7972e8fab3d74383}}

Gets or sets a predicate to determine whether an individual trace request should be accepted. 

The default value for this predicate allows only a subset of traces to be accepted, including the incoming request, outgoing response, and any warnings or errors. If this predicate returns {\ttfamily true} the trace request will be accepted, otherwise it will be discarded. A {\ttfamily null} value for this predicate allows all trace requests.
